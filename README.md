# Highfield Egypt Certificate Verification v9

## Upload to GitHub Pages

Upload the complete contents of this folder to the root of the `verification` repository.

The root should contain:

- index.html
- manifest.json
- service-worker.js
- css/
- js/
- assets/

Do not upload the enclosing `highfield_verification_v9_final` folder itself.

## Current configuration

- Google Apps Script API is configured in `js/config.js`.
- WhatsApp number: +20 110 166 9115.
- Certificate image: `assets/certificate-template.png`.

## Update the phone number

Edit these two values in `js/config.js`:

- whatsappNumber
- whatsappDisplay

## Important

After uploading, wait 1–3 minutes and refresh the GitHub Pages website using Ctrl + F5.
On iPhone, open a private Safari tab if the old cached version still appears.
